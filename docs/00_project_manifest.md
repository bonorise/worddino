# Project Manifest: WordDino (MVP)

## 1. Vision
WordDino 是一个面向非英语母语者的趣味单词探索工具，旨在通过**“认知挂钩（Cognitive Anchoring）”**重构记忆逻辑。
核心隐喻：**“语言考古学”**。
-   **挖掘 (Dig)**：通过拆解词根（挖掘化石），理解单词的底层逻辑。
-   **进化 (Evolve)**：通过谐音和图像（DNA重组），让记忆变得鲜活。
核心理念：
-  **熟悉 + 未知**：利用用户母语中的发音、图像、故事（熟悉）来锚定英语单词（未知）。
-  **结构化记忆**：通过词根词缀家族树，建立单词之间的逻辑联系。

## 2. Branding (The Vibe)
-   **Name**: WordDino
-   **Mascot**: Dr. Dino (一个可爱的恐龙).
-   **Keywords**: Fun, Ancient, Strong, Curiosity.
-   **Slogan**: "Dig up the roots. Master the words." (挖掘词根，征服单词)

## 3. MVP Phase 0 Goals
- **极简验证**：验证“词根拆解 + AI 速记 + 图像”这一核心体验流程。
- **无数据库**：数据层采用“静态高频词根库 + 动态 AI 生成”的混合模式。
- **即时反馈**：用户输入 -> 立即展示词根树与速记卡片。

## 3. Target User Flow
1.  用户进入首页，看到极简搜索框。
2.  输入单词（如 "Inspect"）。
3.  系统展示：
    -   **词根树**：展示 `spect` (看) -> `inspect`, `respect`, `expect`。
    -   **速记卡**：展示 AI 生成的谐音故事 + 关联图片。
